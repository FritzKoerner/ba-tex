\section{Reinforcement Learning}
\label{sec:reinforcement_learning}

Die folgenden Grundlagen basieren auf \cite{suttonReinforcementLearningIntroduction2020}.

\subsection{Allgemein}
\label{subsec:rl_allgemein}

Reinforcement Learning bedeutet zu lernen, welche Handlungen in welchen Situationen auszuführen sind. Die Handlungen sollen dabei eine bestimmte Belohnung maximieren. Der Lerner muss dabei selbstständig herausfinden, welche Handlungen je nach Situation die meiste Belohnung bringen, wobei die Handlungen auch noch zukünftige, nicht nur unmittelbare Belohnungen beeinflussen können.
\\
Die Hauptelemente eines Reinforcement Learning Systems sind: Agent, Umgebung, Strategie, Belohnung, Wert-Funktion und Modell der Umgebung. Die Elemente des Reinforcement Learning werden im folgenden näher erläutert.

\subsection{Markov Decision Process}
\label{subsec:mdp}
 Ein Markov Decision Process (MDP) beschreibt das Problem des Lernens aus Interaktion, um ein bestimmtes Ziel zu erreichen. Reinforcement Learning Probleme können als MDP formuliert werden.

In einem MDP wird der Lerner und Entscheider als Agent bezeichnet. Alles, womit der Agent interagiert, wird Umgebung genannt. Dazu gehört alles, was der Agent nicht willkürlich verändern kann.

Agent und Umgebung interagieren ständig. Nach jeder Aktion präsentiert die Umgebung dem Agenten einen neuen Zustand und eine numerische Belohnung. Der Agent wählt dann auf Basis des Zustands wieder eine neue Aktion. Abbildung~\ref{fig:mdp_agent_environment} zeigt diesen Interaktionszyklus.

\begin{figure}[H]
    \centering
    \includegraphics[width=0.65\linewidth]{Bilder/mdp_agent_environment.pdf}
    \caption{Interaktion zwischen Agent und Umgebung in einem MDP (nach \cite{suttonReinforcementLearningIntroduction2020}, S.~48).}
    \label{fig:mdp_agent_environment}
\end{figure}

Ein MDP ist ein Tripel von Mengen $(\mathcal{S}, \mathcal{A}, \mathcal{R})$, wobei $\mathcal{S}$ Zustände, $\mathcal{A}$ Aktionen und $\mathcal{R}$ Belohnungen sind. Interaktion findet immer zu jedem Zeitpunkt einer Reihe von Zeitpunkten $t = 0, 1, 2, 3, \dots$ statt. Zu jedem Zeitpunkt $t$ beobachtet der Agent einen bestimmten Zustand $S_t \in \mathcal{S}$ und wählt in Abhängigkeit davon eine Aktion $A_t \in \mathcal{A}$ aus. Als Konsequenz erhält der Agent einen Zeitpunkt später eine Belohnung $R_{t+1} \in \mathcal{R}$. Das Zusammenspiel von Agent und Umgebung ergibt folgende Sequenz:
\begin{equation} \label{eq:mdp_sequenz}
    S_0, A_0, R_1, S_1, A_1, R_2, S_2, A_2, R_3, \dots
\end{equation}

Durch MDPs lassen sich Reinforcement Learning Probleme formalisieren. Auf Basis von MDPs lässt sich zeigen das Reinforcement Learning benutzt werden kann um MDPs zu lösen. Während diese theoretischen Grundlagen nur auf endliche MDPs zutreffen, können die Ideen auch auf weiter gefasste Modelle angewandt werden.
